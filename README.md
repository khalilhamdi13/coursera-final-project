# Simple Interest Calculator

This project is a simple interest calculator created as part of the Coursera final project.

## Description
The calculator computes simple interest based on the principal amount, annual rate of interest, and time period.

## Formula Used

Simple Interest = (P × R × T) / 100

Where:
- P = Principal amount
- R = Annual rate of interest
- T = Time period in years

## Features
- Calculates simple interest quickly
- Easy to use
- Written in Bash script

## How to Use

Run the script and enter:
- Principal amount
- Rate of interest
- Time period

The calculator will display the simple interest.

## Example

If:
- Principal = 1000
- Rate = 5%
- Time = 2 years

Then:

Simple Interest = 100
